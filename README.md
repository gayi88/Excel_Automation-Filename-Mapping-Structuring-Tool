# 📊 Excel Automation: Filename Mapping & Structuring Tool

This project automates the process of organizing and inserting filenames into a structured Excel template based on predefined rules.
---

## 🚀 What this project does

This tool:

* Reads data from an input Excel file
* Extracts important information from filenames
* Maps each file to the correct category
* Automatically inserts it into the correct section in another Excel sheet
* Maintains formatting and structure

---

## 📂 Input & Output

### Input:

* Excel file containing raw data (`master.xlsm`)

### Output:

* New Excel file (`master_output.xlsm`)
* Properly structured and updated automatically

---

## Key Features

* ✅ Automatic classification of files
* ✅ Supports multiple sheet types (Station, BEST & Anläggning)
* ✅ Detects correct placement dynamically
* ✅ Creates missing sections automatically
* ✅ Prevents duplicate entries
* ✅ Applies formatting (highlight + structure)
* ✅ Works with large datasets
